## 📝 Описание

Автоматизированный бот для мобильной браузерной игры **"Небоскребы"** (vnebo.mobi). Приложение позволяет управлять несколькими игровыми аккаунтами одновременно, автоматизируя рутинные игровые действия.

## 🎮 Возможности

### Основной функционал
- **Многоаккаунтность**: Управление до 50 аккаунтов одновременно
- **Авторизация**: Поддержка различных платформ:
  - vnebo.mobi (основная)
  - Одноклассники (odkl.vnebo.mobi)
  - Mail.ru (mm.vnebo.mobi)
  - Fotostrana (fs.vnebo.mobi)
- **Табличный интерфейс**: Каждый аккаунт в отдельной вкладке
- **Автоматические обновления**: Проверка и установка новых версий

### Игровая автоматизация
- **Управление жителями**:
  - Выселение по времени (15, 30, 45, 60 минут)
  - Выселение по цвету статуса (зелёные, голубые, жёлтые, фиолетовые, оранжевые)
  - Выборочное выселение (все кроме определённого цвета)

- **Действия со зданием**:
  - Открытие 10-й двери
  - Постройка этажей
  - Позвать в лифт (включая платную опцию за 15)

- **VIP функции**:
  - Отмена коллекции
  - Отмена VIP статуса
  - Управление видимостью на странице "Без города"

### Настройки
- **Интервалы повторов**: Настраиваемые интервалы выполнения действий (в минутах)
- **Персональные задания**: Автоматизация выполнения заданий
- **Сохранение конфигурации**: Все настройки сохраняются в INI файлах

## 🚀 Установка и запуск

### Системные требования
- Windows 7 или выше
- .NET Framework 4.8
- Internet Explorer 11 (для встроенного браузера)
- Активное интернет-соединение

### Запуск
1. Скачайте последнюю версию из [Releases](../../releases)
2. Распакуйте архив в отдельную папку
3. Запустите `vnebo.mobi.bot.exe`
4. Добавьте новый аккаунт через интерфейс
5. Введите данные для авторизации
6. Настройте необходимые параметры
7. Нажмите "ЗАПУСТИТЬ БОТА"

## 🛠️ Технические детали

### Архитектура
```
vnebo.mobi.bot/
├── MainForm.cs          - Главная форма приложения
├── Library/
│   ├── BotEngine.cs     - Основная логика бота
│   ├── HelpMethod.cs    - Вспомогательные методы
│   ├── IniFiles.cs      - Работа с настройками
│   ├── TabDragger.cs    - Перетаскивание вкладок
│   └── Logger.cs        - Система логирования
├── SettingsForm.cs      - Форма настроек
├── Form2.cs             - Дополнительные формы
└── Web.cs               - Работа с веб-запросами
```

### Основные компоненты
- **BotEngine**: Движок бота, обработка игровых действий
- **HttpClient**: Асинхронные HTTP-запросы к серверу игры
- **CancellationToken**: Управление многопоточностью и остановкой ботов
- **IniFiles**: Чтение/запись конфигурационных файлов
- **Logger**: Журналирование событий и ошибок

### Используемые технологии
- **WinForms**: Графический интерфейс
- **async/await**: Асинхронное программирование
- **HttpClient**: Сетевые запросы
- **Regex**: Парсинг HTML-ответов
- **Resources**: Управление изображениями и аватарами

## 📋 Использование

### Добавление аккаунта
1. Нажмите кнопку "Добавить аккаунт" (или используйте меню)
2. Выберите платформу (vnebo.mobi, Одноклассники, Mail.ru, Fotostrana)
3. Введите логин и пароль
4. Настройте параметры автоматизации

### Управление ботом
- **Запуск**: Кнопка "ЗАПУСТИТЬ БОТА" в каждой вкладке
- **Остановка**: Кнопка "ОСТАНОВИТЬ БОТА"
- **Мониторинг**: Статус отображается в реальном времени
- **Логи**: Все действия записываются в лог-файл

### Настройка автоматизации
- Установите интервалы повторов действий
- Выберите типы жителей для выселения
- Настройте выполнение персональных заданий
- Включите/выключите автоматические действия

## ⚙️ Конфигурация

Настройки сохраняются в INI-файлах в папке программы:
- Логины и пароли (в зашифрованном виде)
- Интервалы действий
- Параметры автоматизации
- Настройки интерфейса
